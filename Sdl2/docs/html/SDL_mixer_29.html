4.3.8 Mix_Resume

void Mix_Resume(int channel)

channel
Channel to resume playing, or -1 for all channels.

Unpause channel, or all playing and paused channels if -1 is passed in.

 
// resume playback on all previously active channels
Mix_Resume(-1);
